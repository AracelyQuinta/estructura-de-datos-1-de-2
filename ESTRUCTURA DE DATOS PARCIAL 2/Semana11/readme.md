# Traductor básico Español - Inglés
Este proyecto lo hice porque quería practicar C# y trabajar con diccionarios. La idea es tener un traductor muy simple que pueda tomar frases en español y convertir las palabras que estén registradas al inglés.

## ¿Qué hace?
- Tiene un diccionario inicial con algunas palabras en español y su traducción al inglés.
- Permite agregar nuevas palabras al diccionario mientras el programa está corriendo.
- Traduce frases completas, pero solo cambia las palabras que existen en el diccionario (las demás se quedan igual).
- Muestra un menú interactivo para elegir entre traducir, agregar palabras o salir.
