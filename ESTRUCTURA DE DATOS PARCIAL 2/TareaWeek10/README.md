# Campaña de Vacunación COVID-19 - Informe Ficticio

Este proyecto en **C#** simula la gestión de datos ficticios de ciudadanos en una campaña de vacunación contra el COVID-19.  
El objetivo es aplicar **teoría de conjuntos** para clasificar ciudadanos según su estado de vacunación.

---

## ¿Qué hace el programa?
1. Genera un conjunto de **500 ciudadanos ficticios** con nombres como "Ciudadano 1", "Ciudadano 2", etc.
2. Crea dos subconjuntos:
   - 75 ciudadanos vacunados con **Pfizer**.
   - 75 ciudadanos vacunados con **AstraZeneca**.
3. Aplica operaciones de conjuntos para obtener:
   - Ciudadanos **no vacunados**.
   - Ciudadanos con **ambas dosis** (intersección).
   - Ciudadanos con **solo Pfizer**.
   - Ciudadanos con **solo AstraZeneca**.
4. Muestra los resultados en consola con un formato de informe.
